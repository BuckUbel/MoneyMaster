# Projectmanager V4

## Development

Technologies:   
   * Node.js 
   * Express
   
   + React
   + Material UI

The node server will return the required client-side files.

#### Installation

    yarn install

#### Creating a Dev-Build continuosly

    yarn run start

#### Run the node server

    yarn run server

##### Create a Production build

    yarn run buildProd

##### Create a Dev-Build

    yarn run buildDev
